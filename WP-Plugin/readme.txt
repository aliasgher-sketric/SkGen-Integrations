Integrate the SketricGen AI chat assistant into your WordPress website.

== Description ==

This plugin allows you to easily add the SketricGen AI chat widget to your WordPress website. The widget provides an interactive chatbot experience powered by SketricGen's advanced AI.

Features:
* Easy installation and configuration
* Customizable chat widget
* Powered by SketricGen's AI technology
* Real-time chat responses

== Installation ==

1. Upload the `sketricgen-chat-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > SketricGen Chat to configure the widget
4. Enter your Assistant ID from your SketricGen dashboard
5. Check the "Enable Widget" option and save changes

== Frequently Asked Questions ==

= Where do I get my Assistant ID? =
You can find your Assistant ID in your SketricGen dashboard after creating an AI assistant.

= Can I customize the appearance of the widget? =
Basic customization options are available in the settings page, with more advanced options coming in future updates.
